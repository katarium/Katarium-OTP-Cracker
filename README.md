# 🎯 Descripción

**Katarium OTP Cracker** es una herramienta creada con fines educativos para estudiar cómo pequeñas diferencias en los tiempos de respuesta de una aplicación pueden convertirse en un canal lateral de información (*timing side-channel*).

Su objetivo es ayudar a comprender este tipo de vulnerabilidades y facilitar pruebas de seguridad en entornos controlados o autorizados.

> ⚠️ **ADVERTENCIA LEGAL:** Esta herramienta es **SOLO PARA FINES EDUCATIVOS Y DE INVESTIGACIÓN**. El uso no autorizado contra sistemas de terceros puede ser ilegal. El autor no se responsabiliza por el uso indebido.

---

# 🔬 Metodología

| Fase | Descripción |
|------|-------------|
| 1️⃣ Calibración | Obtiene el tiempo promedio de respuesta del servidor. |
| 2️⃣ Análisis | Compara variaciones de latencia entre respuestas. |
| 3️⃣ Evaluación | Busca diferencias estadísticamente significativas. |
| 4️⃣ Verificación | Repite mediciones para reducir falsos positivos. |
| 5️⃣ Resultados | Muestra métricas obtenidas durante el análisis. |
